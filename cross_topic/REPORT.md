## 一、研究方法

### 1.1 Pipeline 架構

```
Stage 1: 載入語料 → opencc s2t 統一繁體
Stage 2: Qwen3-VL-Embedding-2B 嵌入 (2048 維)
Stage 3: 各語料庫獨立 BERTopic 建模 (KMeans + jieba c-TF-IDF)
Stage 4: Cross-apply — 將 A 的模型套用到 B 的文件上 (BERTopic transform)
Stage 5: 計算 pairing strength + 同主題 BM25 語言差異
```

### 1.2 關鍵設計決策

| 問題 | 解法 |
|------|------|
| 簡繁差異造成假陽性 | opencc s2t 統一所有文本為繁體 |
| BERTopic 在 Python 3.14 會清除 CJK | Monkey-patch `_preprocess_text` |
| Qwen3-VL float16 overflow | 轉 float32 + nan_to_num + L2 normalize |
| 各語料庫主題數不同 | 每源獨立建模，cross-apply 投射 |

## 三、跨語料庫主題投射（Cross-Apply）

### 3.1 方法說明

將語料庫 A 的 BERTopic 模型套用到語料庫 B 的文件上，觀察 B 的文件在 A 的主題空間中如何分布。若 B 的某個 native topic 有 >50% 的文件被投射到 A 的同一個 topic，則認為兩者「對齊」。

**結果**：共發現 299 個強對齊（strength > 0.4），其中 85 個涉及政治議題。

### 3.2 核心發現：同一議題的語言差異

#### 發現一：「兩岸政治」主題的稱謂差異

當各媒體討論同一個「兩岸/中國」議題時：

| 媒體 | 稱呼中國 | 稱呼台灣政府 | 稱呼兩岸關係 |
|------|---------|-------------|-------------|
| **國台辦** | （不需稱呼） | 「當局」「民進黨」 | 「兩岸關係」「融合」 |
| **環球網** | 「中國」「中方」 | 「賴清德」「立法機構」 | 「兩岸」「和平統一」 |
| **旺報** | 「大陸」「北京」 | 「政府」「執政」 | 「兩岸」「交流」 |
| **中時** | 「大陸」「中國大陸」 | 「民進黨」「賴清德」 | 「兩岸」「中華民國」 |
| **匯流** | 「中國大陸」 | 「政府」「總統」 | 「兩岸」「和平」 |
| **自由時報** | **「中國」「中共」** | 「我國」「台灣」 | **「主權」「民主」** |
| **大紀元** | **「中共」**（永遠） | 「台灣」 | 「中共威脅」 |

#### 發現二：國台辦的獨佔詞彙

以下詞彙幾乎只出現在國台辦語料中，台灣媒體（包括最親中的旺報）都不使用：

- **「同胞」**：國台辦在所有主題中反覆使用，台灣媒體幾乎不用
- **「融合」**：國台辦的核心政策詞，台灣媒體偶爾引用但不主動使用
- **「箇中」**：國台辦特有的文言用法
- **「頑固」**：用於「頑固台獨分裂勢力」
- **「廣大」**：用於「廣大台灣同胞」

#### 發現三：中時 vs 國台辦 — 同議題不同框架

**中時社論 T3**（兩岸, 中華民國, 交流, 大陸）投射到**國台辦 T0**（兩岸, 發展, 同胞, 交流），strength=0.944：

| | 中時獨有 | 國台辦獨有 |
|--|---------|-----------|
| 稱謂 | 中華民國, 大陸民眾 | 同胞, 箇中 |
| 人物 | 賴清德 | （不點名） |
| 態度 | 偽政權, 停刊 | 堅持, 頑固, 勢力 |

**解讀**：中時用「中華民國」框架批評民進黨，國台辦用「分裂勢力」框架。兩者立場相近但語言系統完全不同。

#### 發現四：自由時報 vs 國台辦 — 對立框架

**自由時報 T0**（中國, 中華民國, 中共, 和平）投射到**環球網 T9**（日本, 美國, 中國），strength=0.857：

| | 自由時報獨有 | 環球網獨有 |
|--|------------|-----------|
| 核心詞 | 中華民國, **中共**, 出訪, **主權**, 民主 | 大熊, 毛里求斯, 制裁, 出口 |

**解讀**：自由時報在國際議題中強調「主權」「中共」，環球網則完全不觸及台灣主權問題。

#### 發現五：大紀元的「中共」標記

大紀元 T0 被投射到國台辦的**所有 9 個主題**（strength 0.83-1.0），因為大紀元討論的所有中國相關議題都以「中共」開頭，而國台辦的所有主題都涉及中國。

BM25 差異一致顯示：
- 大紀元獨有：**中共**, 軍隊, 控制, 迫害
- 國台辦獨有：兩岸, 融合, 同胞, 交流

## 五、方法論限制

1. **樣本量不均**：國台辦 72 篇 vs 其他 100 篇，自由時報 77 篇（SPA 難爬）
2. **主題粒度**：每源 9-12 個主題，部分主題混雜（如 ETtoday 的「賞屋」廣告混入政治主題）
3. **Cross-apply 的非對稱性**：A→B 和 B→A 的結果不同，需雙向驗證
4. **時間範圍**：2023-2026，部分事件（如金廈翻船）可能過度影響小語料庫
5. **BERTopic 在 Python 3.14 的相容性問題**：需 monkey-patch，未來應固定版本

## 六、結論

### 核心發現

1. **國台辦的語言是完全獨立的系統**：即使統一繁簡後，「同胞」「融合」「當局」「分裂」等詞彙仍為國台辦獨佔，台灣最親中的旺報也不使用。

2. **「大陸」vs「中國」vs「中共」是最強的政治立場指標**：
   - 用「大陸」→ 藍營（旺報、中時）
   - 用「中國」→ 中立或綠營（ETtoday、自由）
   - 用「中共」→ 反共（大紀元）

3. **旺報是「最接近國台辦」的台灣媒體**，但語言系統仍完全不同：旺報用「中華民國」框架，國台辦用「統一」框架。

4. **大紀元和國台辦是光譜兩端**：討論完全相同的議題，但一個永遠加「中共」，一個永遠加「同胞」。

5. **ETtoday 的政治主題被廣告稀釋**：「賞屋」「提供」等商業詞彙混入，反映其商業化程度。
